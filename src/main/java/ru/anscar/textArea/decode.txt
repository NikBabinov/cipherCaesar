Java — универсальный объектно-ориентированный язык со строгой типизацией.

Некоторые достоинства Java:

Простота. У языка чёткие синтаксические правила и понятная семантика.
Объектно-ориентированный подход. В центре внимания находятся данные (объекты), интерфейсы и алгоритмы вторичны.
Безопасность. Разработчики проделали большую работу по защите платформы Java.
Производительность. Новые версии динамических компиляторов Java не уступают традиционным из других платформ.
Надёжность. Программы на Java стабильно работают в любых условиях.
Динамичность и адаптируемость. При необходимости можно добавить в библиотеки новые объекты, методы.
Удобные и эффективные сетевые возможности. Приложения умеют находить нужные объекты в сети и открывать к ним доступ.
Java востребован в таких направлениях, как:

веб-разработка (масштабные бизнес-проекты);
программы для ПК (десктопный софт);
компьютерные игры (например, Minecraft);
приложения для мобильных устройств (ОС Android);
научные исследования и разработки;
промышленный программинг.
